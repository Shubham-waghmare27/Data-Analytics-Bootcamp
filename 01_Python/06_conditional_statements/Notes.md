# 📘 Conditional Statements in Python

## 📖 What are Conditional Statements?

Conditional statements allow a program to make decisions based on a condition.

The program executes different blocks of code depending on whether a condition is **True** or **False**.

---

# 🎯 Why Do We Use Conditional Statements?

- Make decisions
- Execute code based on conditions
- Build real-world applications
- Control program flow

---

# Types of Conditional Statements

- if
- if...else
- if...elif...else
- Nested if

---

# if Statement

```python
age = 20

if age >= 18:
    print("Eligible")
```

---

# if...else

```python
marks = 50

if marks >= 35:
    print("Pass")
else:
    print("Fail")
```

---

# if...elif...else

```python
score = 82

if score >= 90:
    print("A")
elif score >= 75:
    print("B")
else:
    print("C")
```

---

# Nested if

An `if` statement inside another `if`.

---

# Best Practices

- Keep conditions simple.
- Use proper indentation.
- Avoid deep nesting.
