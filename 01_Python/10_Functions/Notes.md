# 📘 Functions in Python

## 📖 What is a Function?

A function is a reusable block of code that performs a specific task.

Instead of writing the same code multiple times, we write it once inside a function and call it whenever needed.

---

# 🎯 Why Do We Use Functions?

- Reuse code
- Improve readability
- Reduce code duplication
- Make programs easier to maintain
- Break large problems into smaller parts

---

# Function Syntax

```python
def function_name():
    # code
```

---

# Calling a Function

```python
greet()
```

---

# Parameters

Parameters are variables defined in the function declaration.

```python
def greet(name):
```

---

# Arguments

Arguments are actual values passed to the function.

```python
greet("Shubham")
```

---

# Return Statement

The `return` statement sends a value back to the caller.

```python
def add(a, b):
    return a + b
```

---

# Default Parameters

```python
def greet(name="Guest"):
    print(name)
```

---

# Best Practices

- Use meaningful function names.
- Keep functions small and focused.
- Prefer returning values over printing when appropriate.

---

# Common Mistakes

- Forgetting to call the function.
- Forgetting the `return` statement when a value is needed.
- Passing the wrong number of arguments.
