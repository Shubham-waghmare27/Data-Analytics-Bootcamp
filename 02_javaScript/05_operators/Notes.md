# Equality Operators

Loose Equality

```javascript
10 == "10"
```

Strict Equality

```javascript
10 === "10"
```

Always prefer **`===`** in modern JavaScript because it compares both value and data type.

# Array Membership

JavaScript does not have `in` like Python for arrays.

Use:

```javascript
languages.includes("Python")
```

---

# Best Practices

- Prefer `===` over `==`.
- Use meaningful variable names.
- Use `.includes()` to check array membership.

---

# Common Mistakes

- Using `=` instead of `===`.
- Confusing `==` with `===`.
- Forgetting `.includes()` for arrays.
